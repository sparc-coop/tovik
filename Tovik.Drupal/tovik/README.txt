Tovik enables automatic, high-quality translation for your Drupal site in minutes.

== Description ==

Tovik is a simple plugin that adds the Tovik automatic site translator to your Drupal site. Instantly make your website multilingual and accessible to a global audience with no coding required.

**Features:**
* Instantly translates your entire site into multiple languages
* Easy setup—-just activate the plugin
* No configuration required
* Powered by Tovik (https://tovik.app)

== Installation ==

1. Log in to your Drupal admin panel.
2. Go to Extend → Install new module.
3. Upload the `tovik` folder and click Install.
4. Return to Extend, find Tovik, check the box and click Install.
5. That's it! Tovik will automatically enable translation on your site.

== Frequently Asked Questions ==

= Do I need to configure anything? =  
No configuration is required. Just activate the plugin.

= Is Tovik free? =  
Visit [https://tovik.app](https://tovik.app) for details about plans and features.

= Where can I get support? =  
For support, visit [https://tovik.app](https://tovik.app).

= How do I uninstall Tovik? =  
In the admin panel, go to Extend → Uninstall. Find Tovik, check the box and click Uninstall. That’s it — the module is removed
